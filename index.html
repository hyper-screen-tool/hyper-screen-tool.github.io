      About the model

      This tool implements the elastic-net penalized logistic regression model used for external validation. Candidate predictors underwent backward stepwise selection, after which the model was trained on three derivation cohorts (CAFPINT, PALI, REDVENT) with α = 0.15 and positive-class weight = 5.5.

      The displayed risk score is raw model output on the weighted training scale (class weight 5.5). Patients with score ≥ 0.70 are classified hyperinflammatory; values below 0.70 are hypoinflammatory. A separate Platt-recalibrated probability is shown for absolute-risk interpretation because class weighting shifts the raw score scale; that calibrated value is not used for the bedside threshold.

      Model development & validation (R)

      Missing data

      Missing PRISM/PELOD physiology values are imputed using published healthy-range midpoints. Weight and other non-PRISM predictors use derivation-cohort medians, matching the primary manuscript analysis. Imputed clinical fields are listed in the results panel. Age is required and is not imputed.

      Model coefficients are available in coefficients.json.

      Clinical disclaimer

      HYPER-SCREEN is a research decision-support tool and is not a substitute for clinical judgment. Predictions should be interpreted in the context of the full clinical picture. For research use and screening enrichment only.
