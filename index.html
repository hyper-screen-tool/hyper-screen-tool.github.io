      About the model

      This tool implements the final elastic-net penalized logistic regression model from our multi-cohort machine learning study. The model was derived on three cohorts (CAFPINT, PALI, REDVENT) using backward stepwise selection followed by elastic-net regularization (α = 0.15, positive-class weight = 5.5).

      Patients with predicted probability ≥ 0.70 are classified as hyperinflammatory; values below 0.70 are classified as hypoinflammatory.

      Missing data

      Missing predictors are replaced with the median value from the combined development dataset, matching the imputation strategy used during model training. Imputed fields are listed in the results panel.

      Model coefficients are available in coefficients.json.
